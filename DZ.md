# Подсказка по GIT

## Инициализация Git в папке
Инициализация git в папке происходит с помощью команды:
```sh
git init
```
## Просмотр статуса Git
Просмотр статуса Git происходит с помощью команды:
```sh
git status
```
git add 

git commit -m "Text"

git checkout

git checkout --oneline

git checkout master

git log

cd

pwd
